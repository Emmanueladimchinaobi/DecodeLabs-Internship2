# Grid & Flexbox Layout 

This project demonstrates how to combine CSS Grid and Flexbox to create a responsive web page layout.

## Layout Structure

The page is divided into the following sections:

- Header
- Sidebar
- Main Body
- Footer

### CSS Grid (The House)

CSS Grid is used to create the overall page structure:

```
+-------------------------+
|         Header          |
+------+------------------+
|      |                  |
|Side  |    Main Body     |
|bar   |                  |
+------+------------------+
|         Footer          |
+-------------------------+
```

### Flexbox (The Furniture)

Inside the Main Body, Flexbox is used to arrange content cards horizontally.

Each card contains:

- Image
- Text
- Button
